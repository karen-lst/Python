1.一行代码实现1--100之和
`print(sum(range(1,101)))`
可以用内置函数sum()实现，sum()里面的参数是可迭代对象，将可迭代对象的元素进行累加。可以是列表、元组、集合类型。
```py
sum([1,2,3])
sum((1,2,3))
sum({1,2,3})
```
