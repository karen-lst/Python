##### 列表的切片
```
      -6   -5   -4   -3   -2    -1
list['AG','IG','WE','QG','OMG','RNG']
       0    1   2     3    4     5
       
list[0:3]: 取第0、1、2上的元素，不包括第三个['AG','IG','WE']

list[-1]:取最后一个元素['RNG']
list[1:]:取除首字母外的所有元素['IG','WE','QG','OMG','RNG']
       
```

### 数据类型问题
```python
a = eval(input())
b = pow(a,3)
print("{:-^20.0f}".format(b))
#input()不管用户输入的是数字还是字符串，它读取的都是一个字符串类型。
#type(a)-->"class str"。
#而pow()的参数数据类型要一致，如果a是str类型就会报错。所以在input()外加一个eval()，去除首尾的引号。a就是数字类型了。
```
